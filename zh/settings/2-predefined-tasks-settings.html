# 2. 预设设置

预设设置最近才来到了wnr。作为默认时间设置的替代品，它让你可以同时保存多组计划。

# 默认预设

在wnr中，有两种预设：默认预设、普通预设。

默认预设会被自动在首页被填充和使用，你还可以在全局设置中启用“启动时自动开始默认预设”的选项，直接开始默认预设，拒绝摸鱼。默认预设在设置中显示为红色。

默认预设不能被删除。



# 普通预设

蓝色的是普通预设。


普通预设和默认预设的功能同样，你还可以把它们设为默认或是删除。

在首页中点开“预设任务”可以选择它们。
